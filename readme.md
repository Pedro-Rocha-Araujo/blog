# Projeto Blog ✏️


### O presente repositório hospeda um projeto de blog feito com as seguintes tecnologias:
* HTML5 | CSS3 -> Usadas para fazer a interface responsiva e visualmente atrativa
* JavaScript -> Usado para adcionar interação à página
* Templates ejs ->  Reaproveitamento de código
* NodeJs | Express -> Interligar o frontent ao banco de dados
* MongoDb | Mongoose -> Usados para armazenar os dados usados na página
* Git | GitHub -> Usados para fazer o versionamento do projeto


### Funcionalidades
* Adcionar um post que poderá ser visualizado na página principal com limitação de caracteres
* Clicar em ler mais, para obter as informações completas correspondentes ao post desejado
* Clicar no ícone de lixeira para deletar o post desejado


### Como usar:
1- Escolher uma pasta para baixar as pastas do projeto

2- Digitar: git clone https://github.com/Pedro-Rocha-Araujo/blog.git

3- Baixar as dependências do projeto: npm i

4- Digitar: node index.js estando na raiz do projeto


### Requisitos para rodar este projeto:
* Ter o nodeJs instalado
* Ter o mongoDb instalado
